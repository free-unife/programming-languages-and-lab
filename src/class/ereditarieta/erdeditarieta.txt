Uomo eredita le proprietà delle superclassi.
Altro esempio e' quello dei veicoli.
Quando una classe eredita qualcosa da un'altra classe si dice che la sottoclasse estende (extends) la superclasse. Si può anche dire che c'è una specializzazione. 

Bisogna compilare i file in ordine dai file Superclasse ai file sottoclasse
